Desenvolvimento de Aplicativo Android / Projeto Próprio
Set 2024 - Nov 2024

fooTTeams - Jogo Equilibrado

Desenvolvido com React Native e Typescript, este aplicativo foi criado tendo em vista sortear equipes para jogos de futebol entre amigos, de modo deixar os times equilibrados. Você cadastra todos os jogadores que costumam jogar, colocando nome, posição preferencial, nota de habilidade para a posição preferencial (1 a 10), posição secundária (posição na qual ele pode jogar, caso necessário), atribuindo-lhe, também, uma nota de habilidade para esta posição. Com os jogadores cadastrados, na hora do jogo, basta selecionar todos os jogadores que compareceram e fazer o sorteio. Antes do sorteio, você ainda seleciona o número de times e a quantidade de laterais, zagueiros, meio-campistas e atacantes para cada time. O algoritmo do aplicativo divide os jogadores de modo a equilibrá-los, afinal assim é muito mais divertido. E não se preocupe caso haja mais jogadores do que vagas nos times, pois após formar todos os times o algoritmo divide também os jogadores restantes como reservas. Além disso, o aplicativo permite exportar e importar listas de jogadores, então diferentes integrantes do/da racha/pelada podem ter a mesma lista de jogadores em seus aplicativos sem a necessidade de cadastrar os jogadores em cada aparelho. É isso, prepare a chuteira e bora brigar pela artilharia!!!!
